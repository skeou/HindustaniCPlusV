# Hindustani C+V

This reclist covers the standard phonetic inventories of pluricentric Hindustani (Urdu and Hindi) in C+V format, ideal for baseline experimentation. Includes a comment file with Nastaliq (Urdu) equivalences and a blank aliased base oto.ini.

This reclist uses a phoneme aliasing system based roughly on IPA and Roman Urdu transcriptions. Comment files are intended for use in Recstar and other Unicode-compatible reclist recorders.

Comment files with Devanagari (Hindi) equivalences, IPA equivalences, and English approximations may be included in a future update.

Base configurations for the base oto.ini may be included in a future update.
